## Full Description (max 4000 chars)

```
Typlx Keyboard is an open-source Android keyboard that brings AI-powered grammar and spelling correction directly to your fingertips — no app-switching required.

Tap the "Fix Grammar" button and Typlx will silently correct everything you've typed in the current text field. The fixed text replaces your draft instantly.

WORKS WITH ANY OPENAI-COMPATIBLE API
Typlx connects to any server that speaks the OpenAI chat completions format. That includes:
• OpenAI (GPT-4o, GPT-4o mini, etc.)
• Anthropic Claude via proxy
• Groq, Together AI, Mistral, and other fast inference providers
• Local models via Ollama or LM Studio

You bring your own API key and choose your model. No subscription, no Typlx account required.

PRIVACY-FIRST DESIGN
• Your API token is stored with AES-256 encryption on your device.
• Text is sent only to the API endpoint you configure — never to Typlx servers.
• No analytics, no crash reporting, no data collection.
• 100% open source under the Apache 2.0 license. Read the code yourself.

FEATURES
• Full QWERTY keyboard with letters, numbers, and symbols
• Caps-lock and shift support
• One-tap grammar and spelling correction for text up to 5,000 characters
• Clear error feedback for network issues or misconfiguration
• Light and dark theme support (follows system setting)
• Works in any app: email, messaging, social media, notes, and more

SETUP
1. Open the Typlx Keyboard app.
2. Enter your API URL, model name, and API token.
3. Tap "Enable Typlx Keyboard in System Settings" and follow the prompts.
4. Switch to Typlx Keyboard in any text field.

OPEN SOURCE
Source code and build instructions are available on GitHub. Contributions, bug reports, and feature requests are welcome.
```
